# Sessie 29 jun 2026 — VibeKilled.rip

## Wat is deze sessie gedaan

### 1. Campfire welkomstbericht (host-greeting)
- **`src/components/Campfire.tsx`**: een "🔥 Campfire host"-begroeting verschijnt bovenaan zodra je de campfire binnenkomt.
- **Puur lokaal per bezoeker** — niet naar Redis geschreven, dus geen flood: een 11de joiner triggert niks bij de andere 10.
- Gedraagt zich als een echt bericht: zit in de berichtenstroom (scrollt weg met nieuwe berichten) en verdwijnt sowieso na **30s** (`showWelcome` + timer, start na de "joining"-beat). Tekst staat als losse `WELCOME`-constante.

### 2. Feed-vlaggen groter & duidelijker (van-land → naar-land)
- **`src/components/LiveFeed.tsx`** (cheer/reaction-regel): vlaggen **12px → 22px** met `drop-shadow-md`, en de pijl nu **goud + `font-black`** (🇳🇱 **→** 🇨🇦). De "we're in this together"-connectie springt eruit.

### 3. Stats: publiek opgepompt (sfeer), admin = alleen echt
- **`src/lib/store.ts`**: `getStats()` (publiek) houdt de ambient-pomp in "down now" (homepage blijft levend).
- Nieuwe **`realActive()`**: telt alleen échte devs = `zcount(index, now, +inf) − ambient − local` (zcount negeert verlopen, niet-getrimde entries).
- `getAdminStats()` gebruikt nu `realActive()` voor "down now" → **admin-dashboard toont alleen echte cijfers**. Online/live-in-chat/kills/revived/total devs waren al echt-only.

### 4. Map vliegt live naar nieuwe ÉCHTE down
- **`src/lib/types.ts`**: nieuw `LastKill { id, lat, lng, at, seq }` + optioneel `lastKill` op `GlobalStats`.
- **`src/lib/store.ts`**: `createPin()` (alleen echt — ambient gaat via `writeAmbientDrafts`) schrijft `vk:lastkill`. `getStats()` leest 'm terug (tolerant: upstash kan parsed object óf JSON-string teruggeven → `safeJsonObj`).
- **`src/app/page.tsx`**: effect op `stats.lastKill`. Baseline op eerste load (geen sprong naar oude kills), springt bij een hogere `seq`, slaat eigen pin (`lk.id === myPinId`) en stale (>30s) over → `focusOn()` + 💀-toast "A dev just went down".
- **Illusie intact**: geen ambient-vlag op publieke pins; client kan bots niet filteren. Signaal hangt aan de stats-poll (6s) → "live genoeg".

### 5. Notificatie-permission meteen bij landen
- **`src/lib/notify.ts`**: `landingDismissed()` / `dismissLanding()` (localStorage-vlag, 1× per device).
- **`src/components/LandingNotify.tsx`** (nieuw): kaartje verschijnt direct bij landen ("🔔 Don't miss your people" — campfire-replies, sympathy/Good4U, resurrectie). Hydration-safe via `mounted`-gate. Native OS-prompt op de **Enable-klik** → gesture-safe (Safari/iOS-proof; auto-firen op load blokkeert Safari). `bottom-6 left-3`, `z-[900]`.
- **`src/app/page.tsx`**: `<LandingNotify />` naast `<Toaster />`.

## Beslissingen / notes
