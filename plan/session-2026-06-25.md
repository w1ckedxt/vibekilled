# Sessie 25 jun 2026 — VibeKilled.rip

## Wat is deze sessie gedaan (van 0 → live)
- 🚀 **Greenfield gebouwd**: Next.js 16 + Tailwind v4, gedeployed op **https://vibekilled.rip** (Vercel, manual CLI deploy).
- 🗄️ **Upstash Redis** (Vercel Marketplace) als database — géén Supabase. Env: `KV_REST_API_URL/TOKEN`.
- 🗺️ **Kaart**: react-leaflet + gratis CartoDB dark tiles. 💀 emoji-markers, 🎆 bij resurrection, coral death-burst + gouden vuurwerk (canvas).
- 🪦 **Kill-flow**: provider + hersteltijd + last words + locatie-toggle (privacy: altijd gejitterd/afgerond, land-niveau).
- ⏳ **Anti-abuse**: 1 actieve pin per user (TTL-lock) + **3u cooldown** na resurrection. Rate limits: 30 kills/u, 200 reacties/min, 12 chat/min per IP.
- ❤️ **Reacties**: Sympathy (groen, tijdens down) / Good4U (goud, ná resurrection) / 🤝 "I hear you" in feed (met teller). Auto-seed: 1 handshake bij drop, auto Good4U bij resurrection. Spectaculaire **links-FX** i.p.v. top-toasts.
- 🔥 **Campfire of Hope**: chat **vast aan je pin-kaartje OP de map**, alleen zolang timer loopt ("joining…" → chat → eruit bij resurrection), "X warming up" live count, **alleen berichten vanaf join** (geen backlog). Gemodereerd (geen links/haatspeech, scheldwoorden mogen).
- 🏅 **Medals**: 30+ over kills/ontvangen/gegeven/comments/streak/downtime. Klik = bubbel bij muis. "Share my position" → X.
- 👑 **Vibe Kings** leaderboard (good4u + sympathy).
- 🌍 **Globe of Pain** feed (max 50, quip-cards, klik = fly-to-pin). Provider-filter. **Global View** knop + auto-focus op je eigen gebied.
- 🛠️ **Admin** (`/admin`, token in Vercel env + .env.local): online now, LIVE IN CHAT, totals, providers, landen, 14-dgn grafiek, **journey-feed** (landed→hit→chatted), **live chat-monitor**.
- 📱 Mobiel: compacte header, inklapbare bottom-sheet, chat werkt (pointer-events fix).
- 🧑‍🎤 "powered by CynicalSally" (klikbaar, onder tagline). 💀 favicon.
- 🔧 Devmode lokaal (`VIBEKILLED_DEV=1`): bypass lock + "reset me" (cleart ook reacties). Staat NIET in prod.
