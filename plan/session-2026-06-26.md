# Sessie 26 jun 2026 — VibeKilled.rip

## Wat is deze sessie gedaan

### 1. Stealth admin chat (geen zichtbare "Sally" meer)
- Admin-berichten gaan nu onder een **willekeurige of zelfgekozen dev-alias** i.p.v. `Sally` + host-badge. Ze renderen exact als een gewone deelnemer, zodat bezoekers niet doorhebben dat de host het gesprek seedt.
- `ChatMessage.official` → interne **`staff`**-vlag. De publieke `/api/chat` GET **stript `staff` + `bot`** zodat interne markers nooit naar de client lekken (niet zichtbaar in network inspect).
- Nieuwe helper `providerForName()` → een persona houdt een **stabiele accent-kleur** over al z'n berichten (geloofwaardig).
- Admin-route `/api/admin/chat` POST: optionele `name` (blank = random `devName()`), zelfde moderatie/reserved-name-regels als echte users.
- Admin-dashboard: alias-veld + 🎲 dobbelsteen, subtiel **"you"**-tagje bij je eigen berichten, naam blijft staan na verzenden (persona vasthouden). Card-titel → "seed as a dev".

### 2. Grote ronde "I've been hit" CTA
- `KillButton` herbouwd: grote **ronde** prominente knop met **doodskop over gekruiste zwaarden** (💀 + ⚔️), pulserend, responsive (h-28 mobiel → h-32 desktop).
- Speelse uitnodiging erboven: *"Just got rate-limited? Drop it here, soak up the sympathy, fling some back — then BAM."*
- Modal-CTA → **"BAM — log it 💥"**.

### 3. Map: random refresh/glitch gefixt + begrensde wereld
- **Root cause glitch**: `FocusController` heropende elke 4s-poll de popup van de laatst-gefocuste pin (`pins` in deps, geen guard); Leaflet-popup `autoPan` verschoof daardoor steeds de map. → **`opened`-guard**: popup opent nog maar **één keer per focus-actie**. Dit fixte zowel "verplaatst random" als "meest recente blijft openstaan".
- **Markers gememoized** (`PinMarker` + `React.memo`, stabiele `onRef`-setter, per-marker icon-`useMemo`): onveranderde pin re-rendert niet meer op een poll → geen flikker. `iconCache`-ref tijdens render verwijderd (lint-clean).
- **Begrensde wereld**: `worldCopyJump` eruit, TileLayer `noWrap`, `maxBounds` + `maxBoundsViscosity={1}`. Vrij pannen/zoomen maar duidelijke randen, geen oneindig scrollen, geen 360°-view-sprongen.
- **Jouw kaart is de baas**: auto-follow naar nieuwe pins is **pan-only** (opent nooit een popup) en **uit zolang je zelf down bent**.
- **Auto-follow**: nieuwe binnenkomende kills → zachte pan ernaartoe (`Math.max(zoom,4)`, "net genoeg", zoomt nooit verder uit), slaat de initiële batch over.
- Nieuwe **"🔥 My campfire"**-knop (rechtsonder, als je down bent) → vliegt terug naar je eigen pin + chat. **🌍 Global View** blijft.
- Mobiel: bottom-sheet **start dichtgeklapt** zodat de grote CTA-knop meteen bereikbaar is.
